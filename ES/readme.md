## 現在実施しているゼミ/研究のテーマおよび内容をご記入ください。
VR環境に被験者たちはウェアラブルセンサーをつけてもらって、多人対話による感情喚起の実験をさせる。  
そして、被験者の脳波EEGと心電ECGのデータをとって、自己申告をしてもらう。  
データ前処理後に、SVMを使ってデータ分類を行う。生体情報と感情・対話雰囲気の関係を探す。
